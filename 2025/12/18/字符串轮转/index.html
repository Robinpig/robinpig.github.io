字符串轮转

Description

假定有一种 isSubstring 方法，可检查一个单词是否为其他字符串的子串。
给定两个字符串 s1 和 s2，请编写代码检查 s2 是否为 s1 旋转而成，要求只能调用一次 isSubstring
（比如，waterbottle是 erbottlewat 旋转后的字符串）

Solution

无论 s2 如何旋转, 都是将 s1 的 尾部拼上 头部, 所以可以 使用 s1+s1, 判断 s2是否是 s1s1的子串
